# Cambios v11 — formulario de productos

- El campo ID queda oculto en el formulario; se asigna automáticamente.
- Se ocultan URL de foto, Link de WhatsApp y Fecha de alta en productos; se llenan automáticamente.
- La foto del producto se sube desde el formulario y se guarda en Drive con nombre automático `ID_nombre-de-la-prenda_01.ext`.
- Al subir una nueva foto para un producto existente, se sustituye la anterior con el mismo nombre.
- Productos exige: nombre, descripción, medidas, precio, foto y bordadora registrada.
- Bordadora sigue siendo la fuente central: se selecciona desde la hoja Bordadoras y de ahí se derivan municipio, localidad y WhatsApp.
